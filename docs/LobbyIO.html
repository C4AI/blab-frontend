# getConversations(interval, callback, failCallback, retries)

Get the list of current conversations.

Since this method should only be used in a development environment, it only returns the first page of results.

Parameters:
Name Type Description
interval number

interval between server requests to get the conversation list (or null to request only once)

callback conversationListCallback

function called when the conversation list is refreshed

failCallback errorCallback

function called if the request fails (after retries, if any)

retries number

number of retries if the request fails with 5XX
